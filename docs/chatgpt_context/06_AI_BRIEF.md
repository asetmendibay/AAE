# AI Brief

## Automation Engine

- **Версия:** Project Baseline 0.1
- **Текущий этап:** Этап 1 — Engineering Foundation (в процессе перехода)
- **Текущая задача:** Создание и интеграция ChatGPT Context Layer (`docs/chatgpt_context/`) для восстановления архитектурной памяти.
- **Последние принятые решения:**
  - Создание отдельной папки `docs/chatgpt_context/` для восстановления состояния ChatGPT без чтения всей кодовой базы.
  - Назначение ChatGPT роли Independent Architecture Advisor (канонический контекст предоставляется Codex).
- **Открытые вопросы:**
  - Определение точных контрактов логирования, конфигурации и таксономии ошибок (должно быть согласовано перед Этапом 2).
- **Следующая задача:** Инициализация scaffolding проекта на Node.js/TypeScript, настройка ESLint, Prettier, test runner и базового CI для Этапа 1.
- **Основные риски:**
  - Накопление security debt (логирование конфиденциальных данных, небезопасное хранение сессий/профилей браузера) до реализации механизмов защиты.
  - Отклонение от baseline-архитектуры при переходе к реализации (контролируется правилом Architecture Freeze).
- **Последнее обновление:** 2026-07-18
