# Executive Context

**Назначение:** Краткий обзор проекта и текущей фазы разработки для восстановления контекста ChatGPT.

## 1. Описание проекта (Что такое проект)

Automation Engine (AAE) — долгосрочный проект по разработке модульной, расширяемой и контролируемой платформы разрешённой браузерной автоматизации на Node.js, TypeScript и Playwright. Платформа создаётся как альтернатива разрозненным скриптам автоматизации и закрытым решениям вроде Browser Automation Studio.

## 2. Текущий статус

- **Текущая версия:** Project Baseline 0.1 и Engineering Foundation.
- **Текущий этап:** Этап 7 — System Verification.
- **Текущая цель:** Проверить полный локальный runtime через fault-injection и security review.
- **Завершено:** Config, Logger/Events, Error taxonomy, Execution Engine, Browser Platform, Runtime Resources и Local Smoke module.
