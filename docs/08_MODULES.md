# Modules (Модули системы)

В данном документе описывается структура, интерфейсы и взаимосвязи всех активных программных модулей движка Automation Engine.

> [!NOTE]
> На текущем этапе (Фаза 1: Проектирование) программные модули находятся в состоянии разработки архитектурных спецификаций. После утверждения архитектуры каждого модуля его интерфейс и зависимости будут документироваться здесь.

## Ожидаемые модули ядра (Фаза 1)
1. **Config Module** — управление конфигурацией приложения (валидация окружения, парсинг настроек).
2. **Logger Module** — ведение логов работы системы с поддержкой уровней логирования и разделением по потокам.
3. **Browser Manager** — управление процессами браузеров Playwright, подмена параметров отпечатков.
4. **Proxy Manager** — загрузка, распределение, валидация и ротация прокси-серверов.
5. **Profile Manager** — загрузка, хранение, шифрование сессий (cookies, localStorage, indexedDB).
6. **Fingerprint Manager** — генерация реалистичных конфигураций браузера и инжекция параметров эмуляции.
7. **Data Generator** — генератор случайных профильных данных пользователей (имена, пароли, юзер-агенты).
8. **CLI Controller** — интерфейс командной строки для контроля выполнения задач и потоков.
