# API Specification (Спецификация API)

В данном документе документируются внешние интерфейсы взаимодействия с Automation Engine.

## Типы интерфейсов
1. **CLI API** — параметры командной строки для запуска сценариев автоматизации.
2. **REST API** — эндпоинты панели управления (Dashboard) для интеграции с внешними системами.
3. **Webhook API** — формат уведомлений, отправляемых движком в сторонние сервисы (например, в Telegram или CRM).
4. **Library API** — публичные TypeScript/JavaScript методы для использования Automation Engine в качестве библиотеки в стороннем Node.js коде.

> [!NOTE]
> Спецификация API будет наполняться по мере реализации соответствующих модулей.
