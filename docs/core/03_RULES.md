# Правила проекта

## Никогда

- ❌ не использовать координаты
- ❌ не писать большие функции
- ❌ не хранить настройки внутри кода
- ❌ не использовать фиксированные sleep без причины
- ❌ не дублировать код
- ❌ не создавать зависимости между модулями без необходимости

## Всегда

- ✔ логировать ошибки
- ✔ писать модульный код
- ✔ использовать конфигурацию
- ✔ писать комментарии там, где это действительно необходимо
- ✔ разделять ответственность модулей
- ✔ думать о масштабировании
