# Automation Engine — Project Vision

**Версия:** 0.1 Baseline
**Владелец продукта:** Aset
**Статус:** действует

## Назначение

Automation Engine (AAE) — собственная платформа разрешённой браузерной
автоматизации на Node.js, TypeScript и Playwright. Она создаётся как
поддерживаемая инженерная альтернатива визуальным средам автоматизации, а не как
набор независимых скриптов.

## Цель

Создать контролируемый модульный движок, в котором новый сценарий автоматизации
подключается к устойчивому runtime, а не дублирует управление браузером,
настройками, логированием, ошибками и жизненным циклом выполнения.

## Принципы продукта

- Модульность, низкая связанность и явные контракты.
- Надёжность, диагностика и восстановление важнее скорости добавления функций.
- Конфигурация, секреты и runtime-данные отделены от кода.
- Сценарии запускаются только для систем и действий, на которые у владельца
  проекта есть разрешение.
- Повторное использование инфраструктуры важнее оптимизации под один сайт.
- Изменение внешнего сайта должно затрагивать конкретный сценарий, а не ядро.

## Граница версии 0.1

Версия 0.1 должна доказать работу локального execution runtime на одном
разрешённом сценарии: конфигурация, структурированные логи, управляемый запуск
задачи, изолированный браузерный контекст, profile storage и проверяемый
результат.

В 0.1 не входят Dashboard, внешний REST API, cloud synchronization, массовые
интеграции провайдеров или функции, требующие обхода ограничений сторонних
систем.
