# Automation Engine

**Версия:** 1.0  
**Автор:** Aset  

## Описание

Automation Engine — это собственный движок автоматизации браузера.  
Проект создается как альтернатива Browser Automation Studio.  
Главная цель — получить полностью контролируемую архитектуру, которую можно расширять практически без ограничений.  

## Основные требования

* ✔ модульность
* ✔ масштабируемость
* ✔ читаемый код
* ✔ высокая стабильность
* ✔ автоматическое логирование
* ✔ повторное использование модулей
* ✔ поддержка множества сайтов
* ✔ работа с профилями
* ✔ работа с прокси
* ✔ работа с fingerprint
* ✔ управление браузерами
* ✔ система восстановления после ошибок
* ✔ минимальные изменения при обновлении сайтов

Проект должен разрабатываться так, чтобы спустя год его было легко поддерживать.
